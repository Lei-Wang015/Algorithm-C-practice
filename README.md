# Algorithm-C-practice

## 用于记录本人在学习c语言过程中的练习
- 网站为https://pintia.cn/problem-sets/dashboard
- 学习方法为：（三循环法）
  - Day1:让AI写出题目的完整解法代码并总结思路，复制思路后根据思路的提示自己写出代码（部分卡住了可以回头看完整解法），不懂的部分一定要问AI(比如为什么读入字符串时不用写取址符)
  - Day2:根据思路提示自己完整写出代码（不要允许看任何提示，自己debug），如果实在写不下去就看看完整解法并将今天重置为Day1
  - Day3:自己写思路和代码，如无思路则将今天重置为Day2
